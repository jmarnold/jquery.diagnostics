    • jquery.diagnostics

    Easy to use diagnostics API for providing JS instrumentation.

    Project maintained by jmarnold Hosted on GitHub Pages — Theme by mattgraham

    Ever needed custom overlays to provide debugging information?

    Think of the shortcut tricks when you hit "?" on Github, Gmail, etc.

    jquery.diagnostics aims to make this easy by providing a "dashboard" and an API for adding screens.

    See it in action

    Simply type "ctrl+d" and you will see the Dashboard load. Type ESC to hide it.
